Help & Documentation

This guide will help you navigate the OSINT platform and understand how to make the most of its features.

Basic OSINT Investigation

Input Data

Start by entering as much information as you have about the subject. All fields are optional, but at least one is required.

AI Processing

Results

Review the comprehensive report with visualizations, connections, and insights from multiple sources.

Workflow System

The workflow system allows you to automate intelligence gathering with configurable steps and triggers.

Workflow Levels
Basic

Simple workflows with limited data points and basic analysis, suitable for beginners or routine checks.

  • Single target focus
  • 1-3 data collection steps
  • Basic report generation
Intermediate

Moderate complexity with multiple data sources and more detailed analysis.

  • Multiple related targets
  • 4-6 data collection steps
  • Pattern detection
  • Detailed reporting
Expert

Complex intelligence gathering with extensive sources and advanced analysis.

  • Multiple target networks
  • 7+ data collection steps
  • Cross-reference verification
  • Anomaly detection
  • Comprehensive intelligence reports
Creating Workflows
  1. Go to the Workflows page
  2. Click Create New Workflow
  3. Enter a name and description
  4. Define the workflow steps:
    • Data Collection: Specify which data to collect and from where
    • Analysis: Configure how the data should be analyzed
    • Reporting: Set up the reporting format and delivery method
  5. Set the workflow trigger (schedule, event, or manual)
  6. Save the workflow
Executing and Monitoring

You can execute workflows manually or set them to run automatically based on schedules or events.

The workflow execution history shows:

  • When each workflow was executed
  • Success or failure status
  • Detailed logs of each step
  • Any errors encountered
  • Results and generated reports

API Configuration

The platform uses various APIs to collect intelligence. You can configure these APIs in the API Configuration section.

RapidAPI Integration

This platform uses RapidAPI as the primary source for OSINT APIs. To get started:

  1. Create an account on RapidAPI
  2. Subscribe to relevant APIs
  3. Get your RapidAPI key
  4. Enter your RapidAPI key in the API Configuration page
Custom API Configuration

You can also add custom APIs to the platform:

  1. Go to the API Configuration page
  2. Click Add New API
  3. Enter the API details including name, URL, and authentication method
  4. Define the endpoints and response format
  5. Save the configuration

The system will then include your custom API in the selection process when determining which APIs to use for investigations.

Ethical Use Guidelines

Please observe the following ethical guidelines when using this platform:

  • Only conduct OSINT on subjects with proper authorization
  • Respect privacy and data protection laws
  • Do not use the platform for harassment or stalking
  • Securely store any sensitive information collected
  • Use the information gathered responsibly and ethically
