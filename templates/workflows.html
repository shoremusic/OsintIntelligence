Workflow Engine

Workflow Engine Status: Unknown

The workflow engine handles scheduled and event-triggered workflows automatically. You must initialize the engine before starting it.

Workflow Help
Step Types:
  • api_query - Query APIs for intelligence data
  • llm_analysis - Analyze data using AI
  • web_scrape - Extract data from websites
  • data_store - Store collected data
  • condition - Branch workflow based on conditions
Trigger Types:
  • manual - Run workflow on demand
  • schedule - Run workflow on a schedule
  • event - Run workflow when new data appears
Example Workflows:

Purpose: Verify email addresses and gather basic information.

[
  {
    "step_type": "api_query",
    "name": "Email Verification",
    "config": {
      "api_name": "EmailVerifier",
      "endpoint": "verify",
      "params": {
        "email": "{{ input.email }}"
      }
    }
  },
  {
    "step_type": "condition",
    "name": "Check Verification Result",
    "config": {
      "condition": "{{ steps.0.result.valid == true }}",
      "true_path": 2,
      "false_path": 4
    }
  },
  {
    "step_type": "api_query",
    "name": "Domain Information",
    "config": {
      "api_name": "DomainInfo",
      "endpoint": "info",
      "params": {
        "domain": "{{ input.email.split('@')[1] }}"
      }
    }
  },
  {
    "step_type": "llm_analysis",
    "name": "Generate Report",
    "config": {
      "prompt": "Analyze the email verification and domain information and provide a brief report.",
      "data": {
        "email": "{{ input.email }}",
        "verification": "{{ steps.0.result }}",
        "domain_info": "{{ steps.2.result }}"
      }
    }
  },
  {
    "step_type": "llm_analysis",
    "name": "Invalid Email Report",
    "config": {
      "prompt": "Generate a report explaining why the email is invalid and provide suggestions for next steps.",
      "data": {
        "email": "{{ input.email }}",
        "verification": "{{ steps.0.result }}"
      }
    }
  }
]

Purpose: Gather and analyze social media profiles across multiple platforms.

[
  {
    "step_type": "llm_analysis",
    "name": "Username Variants",
    "config": {
      "prompt": "Generate 5 likely username variants based on the name and any known usernames.",
      "data": {
        "name": "{{ input.name }}",
        "known_username": "{{ input.social_media }}"
      }
    }
  },
  {
    "step_type": "api_query",
    "name": "Social Media Profile Finder",
    "config": {
      "api_name": "ProfileFinder",
      "endpoint": "search",
      "params": {
        "usernames": "{{ steps.0.result.username_variants }}"
      }
    }
  },
  {
    "step_type": "condition",
    "name": "Check If Profiles Found",
    "config": {
      "condition": "{{ steps.1.result.profiles|length > 0 }}",
      "true_path": 3,
      "false_path": 6
    }
  },
  {
    "step_type": "web_scrape",
    "name": "Scrape Profile Pages",
    "config": {
      "urls": "{{ steps.1.result.profile_urls }}"
    }
  },
  {
    "step_type": "api_query",
    "name": "Image Analysis",
    "config": {
      "api_name": "ImageAnalyzer",
      "endpoint": "analyze",
      "params": {
        "image_urls": "{{ steps.1.result.profile_images }}"
      }
    }
  },
  {
    "step_type": "llm_analysis",
    "name": "Profile Analysis Report",
    "config": {
      "prompt": "Analyze the social media profiles found. Identify patterns, interests, connections, and any significant findings.",
      "data": {
        "input_info": "{{ input }}",
        "profiles": "{{ steps.1.result.profiles }}",
        "profile_content": "{{ steps.3.result }}",
        "image_analysis": "{{ steps.4.result }}"
      }
    }
  },
  {
    "step_type": "llm_analysis",
    "name": "No Profiles Report",
    "config": {
      "prompt": "Generate a report explaining that no profiles were found and suggest alternative approaches.",
      "data": {
        "input_info": "{{ input }}",
        "username_variants": "{{ steps.0.result.username_variants }}"
      }
    }
  }
]
